# 🌐 Network Labs – Cisco Packet Tracer

This repository contains my networklabs, made in Cisco Packet Tracer.

The purpose is to document my practical networking skills within:
- VLAN segmentation
- Routing (static & dynamic)
- Switching fundamentals
- NAT
- Access Control Lists (ACL)
- Network troubleshooting

## 🎓 Labs

| Lab | Description |
|-----|------------|
| VLAN Basics | Network segmentation with VLANs |
| Inter-VLAN Routing | Communication between VLANs via Layer-3 equipment |

## 🛠️ Tools
- Cisco Packet Tracer
- CLI (Cisco IOS)

## 📌 Key Focus
- Hands-on configuration using Cisco IOS CLI
- Troubleshooting real network scenarios
- Understanding network design and segmentation

## 📈 Goals
To build hands-on experience and demonstrate practical networking skills through structured lab environments.
